# [초급편] 안드로이드 커뮤니티 앱 만들기(Android Kotlin)

## 프로젝트 소개

- 자취생에게 필요한 정보 제공
- 게시판을 통해 서로 정보 공유 가능
- 북마크 기능을 통해 원하는 정보 저장 가능
